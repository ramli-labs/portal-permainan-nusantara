# Panduan IFP Offline

1. Sambungkan IFP ke internet.
2. Buka portal melalui Chrome atau Edge.
3. Pilih **Pasang Aplikasi**.
4. Buka Gobak Sodor, Jelajah Nusantara, Congklak Cerdas, Engklek Pintar, dan Egrang Nusantara satu kali.
5. Buka **Cek Offline** sampai seluruh komponen berstatus siap.
6. Putuskan Wi-Fi dan uji sedikitnya satu sesi pada setiap game.
7. Buat PIN pada Mode Guru.
8. Periksa bahwa seluruh hasil muncul di Peringkat dan Riwayat.
9. Lakukan backup JSON secara berkala ke USB.

Data tersimpan pada profil browser/PWA perangkat. Menghapus data browser akan menghapus data permainan jika belum dibackup.
