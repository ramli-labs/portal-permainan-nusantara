# Panduan Pengguna Portal Permainan Nusantara v2.2.0

## Bermain

1. Buka portal.
2. Pilih **Pilih Game**.
3. Pilih salah satu dari lima permainan.
4. Pilih **Solo** atau **Berdua**.
5. Masukkan nama pemain dan kelas/kelompok bila diperlukan.
6. Pilih tingkat kesulitan.
7. Tekan **Mulai Permainan**.
8. Hasil otomatis disimpan setelah permainan selesai.

## Peringkat

Halaman **Peringkat** dapat dilihat siswa. Pada filter Semua Game, portal memakai Indeks Peringkat yang menggabungkan skor, akurasi, dan bonus kemenangan. Saat satu game dipilih, urutan memakai skor asli game tersebut.

## Mode Guru

1. Buka **Guru**.
2. Buat atau masukkan PIN 4–8 angka.
3. Sesi Mode Guru aktif selama 45 menit pada tab tersebut.

### Editor Soal

- Pilih game.
- Buat set baru atau salin bank bawaan.
- Tambahkan minimal enam soal.
- Tekan **Aktifkan**.
- Permainan berikutnya memakai set aktif.
- Gunakan Ekspor JSON untuk memindahkan set melalui USB.

Bank bawaan tidak dapat dihapus dan selalu dapat diaktifkan kembali.

### Statistik

Statistik menampilkan jumlah sesi, pemain, tingkat kemenangan, akurasi, skor rata-rata, kinerja per game, kinerja per pemain, dan rapor kategori.

### Pemain

Guru dapat memperbaiki nama atau kelas. Perubahan nama juga diterapkan pada hasil yang terkait. Menghapus profil beserta sesi tidak dapat dibatalkan tanpa backup.

### Hasil

Halaman Hasil memerlukan sesi PIN Guru. Filter dapat diterapkan sebelum mengekspor CSV. File CSV hanya berisi hasil yang tampil setelah filter.

### Backup

- **Backup JSON:** menyimpan profil, hasil, pengaturan, PIN hash, dan set soal.
- **Restore JSON:** menampilkan ringkasan file sebelum data digabungkan atau diganti.

Simpan backup pada USB secara berkala.
