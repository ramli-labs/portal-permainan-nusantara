# Release v2.1.0 — Complete Games

## Perubahan

- Lima game berstatus tersedia dan dapat dimainkan offline.
- Jelajah Nusantara diperluas menjadi 15 soal, lima kawasan, tiga kesulitan, dan mode Berdua.
- Congklak Cerdas ditambahkan dengan papan 7 lubang, AI, tangkapan, giliran tambahan, dan kuis matematika.
- Engklek Pintar ditambahkan dengan gacuk, jalur petak, ronde bergantian, dan soal campuran.
- Egrang Nusantara ditambahkan sebagai adaptasi ritme dan keseimbangan, termasuk relay Berdua.
- Filter leaderboard dan hasil mendukung seluruh game.
- Diagnostik offline memeriksa lima bank soal.
- Ekspor CSV dilindungi dari formula injection sederhana.
- Nama sesi dirender sebagai teks, bukan HTML.
- Skor game baru dibatasi maksimum 10.000 poin.

## Batas yang perlu dipahami

- Engklek dan Egrang adalah adaptasi digital untuk IFP, bukan simulasi gerak fisik penuh.
- Pengujian sentuhan, ukuran layar, fullscreen, dan kenyamanan dua pemain wajib dilakukan pada IFP target.
- Mode Guru pada versi ini mengelola hasil, PIN, backup, dan pemulihan. Editor bank soal lintas-game belum menjadi bagian release ini.
