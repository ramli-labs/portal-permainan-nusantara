# Desain Game v2.1.0

## Prinsip

- Semua game dapat dimainkan tanpa jaringan setelah aset tercache.
- Tombol sentuh dibuat besar untuk IFP.
- Mode Berdua disesuaikan dengan mekanik permainan, bukan dipaksakan menjadi pola yang sama.
- Setiap game menyimpan satu kontrak hasil yang sama ke leaderboard portal.

## Mode Berdua

| Game | Bentuk Berdua |
|---|---|
| Gobak Sodor | Co-op simultan |
| Jelajah Nusantara | Bergantian menjawab sebagai tim |
| Congklak Cerdas | Berhadapan dan bergantian |
| Engklek Pintar | Bergantian per ronde |
| Egrang Nusantara | Relay, berganti pada 50 meter |

## Batas adaptasi digital

Engklek dan Egrang pada IFP tidak menggantikan aktivitas fisik aslinya. Keduanya merupakan adaptasi digital yang menekankan urutan, koordinasi sentuh, ritme, pengetahuan, dan pengenalan budaya. Praktik permainan fisik tetap memerlukan ruang, alat, pengawasan, dan prosedur keselamatan yang sesuai.
