# Audio

Versi 1.0 menggunakan **Web Audio API** melalui `js/audio.js`. Musik latar dan efek suara dibuat secara prosedural, sehingga proyek tidak membutuhkan file audio eksternal dan tetap ringan untuk mode offline.

Folder ini dapat dipakai bila pengembang ingin mengganti audio prosedural dengan rekaman orisinal berformat `.ogg` atau `.mp3`.
